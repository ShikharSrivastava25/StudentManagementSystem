Developed a Student Management System using ASP.NET Core and InMemory Storage.
Implemented the CQRS + MediatR design pattern.
Used Fluent Validation to validate the POST and PUT requests. 

General Information about the project : 
1. The students table consists of students details and his/her marks.
2. The marks table defines the model which is used to store a students marks.

User Guide : 

1. Initially, use the Post requests on student table to enter a students' information.
2. Use the Post request under Marks endpoints to enter the marks of a student in specific subject.
3. After initial values for student and his/her marks are defined, use other requests like Put, Delete and Get.
